\section{Examples}

Example \ref{conan1} is a numbered conversation analysis example:

\begin{conan}[Anton]
	\alone{Anton}{Lorem ipsum dolor sit amet, consetetur sadipscing elitr, sed diam nonumy eirmod tempor invidunt ut labore et dolore magna }
	\simul{}{aliquyam erat, [ sed diam ] voluptua.}{Zita}{[ ero eos et ] }
	\label{conan1}
\end{conan}


\texttt{conan} environments can easily be mixed with \texttt{example} environments from the \texttt{covington} package. Both use the equation counter for numbering.

\begin{example}
Covington Example
\label{covington1}
\end{example}

The next example is a \texttt{conan*} environment without numbering.

\begin{conan*}

	\simul{}{ erat, [ sed diam ] voluptua.}{Z}{[ ero eos et ] }
\end{conan*}
